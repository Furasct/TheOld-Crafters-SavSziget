Ha nagyobb hibát jelentesz, akkor akár jutalmat is kaphatsz. Jutalom lehet játékpénz, vagy akár OCD. (Több információ az OCD -ről: https://theold.minemarket.hu/)

A jelentésekkor ne használjatok trágár szavakat.
